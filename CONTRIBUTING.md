# 🤝 Contributing Guide

Thank you for contributing to TVbox KB.

## How to Contribute

- Report bugs
- Suggest new features
- Improve documentation
- Submit pull requests

## Pull Request Steps

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Commit your changes
5. Submit a pull request

## Rules

- Keep code clean and readable
- Test changes before submitting
- Do not share secrets, tokens, or passwords
- Be respectful to other contributors

## License

By contributing, you agree that your contributions will be released under the project's license.

Thank you for helping improve TVbox KB ❤️
