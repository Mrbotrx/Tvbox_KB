## 🤝 How To Contribute

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Commit your work
5. Submit a Pull Request

Example:

```bash
git checkout -b feature/my-feature
git commit -m "Add new feature"
git push origin feature/my-feature
```
