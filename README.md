## 📥 How To Use

### Android

1. Open IPTV Player
2. Add Playlist
3. Paste URL
4. Save
5. Enjoy

### Windows

1. Open VLC
2. Media → Open Network Stream
3. Paste Playlist URL
4. Click Play

### Smart TV

1. Install IPTV App
2. Add Remote Playlist
3. Paste Playlist URL
4. Load Channels
